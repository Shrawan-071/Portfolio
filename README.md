## 🖱️ Interactive 3D Background

The portfolio includes a dynamic algorithm-based 3D background that reacts to the user's mouse movement.

The interactive system includes:

- Mouse position tracking
- Dynamic particle movement
- Straight-line connections between nearby nodes
- Cursor path tracing
- Geometric pattern generation
- Depth and parallax effects
- Smooth real-time animations

The background continuously changes and responds to user interaction, creating a living digital network.
